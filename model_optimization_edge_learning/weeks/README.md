# 20-22 Week Curriculum

这套课程分为两个阶段：

- 阶段一：手写原理，重点看清实现机制。
- 阶段二：主流框架，重点看清工程落地。

建议每周 6-10 小时。如果时间不足，可以把每个双周主题拆成 3 周。

每个周任务都包含答案闭环：先自己做，卡住时看提示，再看参考答案、原理解释和自检标准。学习时建议先至少尝试 20-30 分钟，再打开答案块。
