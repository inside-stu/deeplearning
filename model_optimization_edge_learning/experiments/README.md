# Experiments

每次实验复制 `templates/experiment_log.md`，建议命名：

```text
20260616_phase1_week01_handwritten_baseline.md
20260630_phase1_week04_lora_head.md
20260715_phase1_week08_qat.md
20260820_phase2_week12_mmdetection_baseline.md
```

实验记录要能回答：

- 这次实验在验证哪个底层机制？
- 手写实现和框架实现是否一致？
- 指标变化是多少？
- 下一步该保留、放弃还是继续验证？
